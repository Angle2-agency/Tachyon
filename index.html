What is Tachyon?

  • Tachyon ensures privacy through the use of the MimbleWimble Protocol
  • Tachyon is the first privacy-centric cryptocurrency using formal verification, guaranteeing security and correctness
  • Tachyon features Proof of Stake, ensuring a fair distribution of rewards to stakeholders.

Why the Tachyon?

The Tachyon is the solution

The Tachyon took into account the mistakes and disadvantage other cryptocurrencies. So it has superiority over others. Tachyon is an implementation of a private transaction protocol called MimbleWimble.

Ensuring Privacy through MimbleWimble Protocol

  • Privacy

    It features the strongest privacy based on the MimbleWimble protocol. Transaction amount, sender, and receiver information are all hidden and untraceable through homomorphic encryption.This enables complete fungibility without precluding the ability to selectively disclose information as needed.

  • Scalability

    MimbleWimble is very lightweight, requiring little data storage (even less than Bitcoin). Scales mostly with the number of users and minimally with the number of transactions (<100 byte `kernel), resulting in a large space saving compared to other blockchains. Also a technology called bulletproofs allows for increased scalability.

  • Security

    Strong and proven cryptography. MimbleWimble only relies on Elliptic Curve Cryptography which has been tried and tested for decades.

  • Auditability

    Design simplicity that makes it easy to audit and maintain over time.

Formal Verification

  • Correctness

    Tachyon uses the “proofs-as-programs” paradigm, which guarantees correctness due to the Curry-Howard correspondence

  • Privacy

    Tachyon's formally verified implementation will use the Strand Spaces model [Thayer et al 1999]. It allows for powerful additional privacy guarantees.

  • Security

    Tachyon is creating a reference implementation that only uses primitives that have been specified precisely and verified with proof assistants, guaranteeing security.

Delegated Proof of Stake (DPOS)

  • Economics

    Tachyon uses a Delegated Proof of Stake (DPOS) system, which benefits stakeholders. Stakeholders can earn more cryptocurrencies and secure the network in the process.

  • Security

    DPOS systems have security benefits over traditional Proof of Work systems, by requiring miners to deposit cryptocurrency.

  • Community

    Tachyon’s DPOS allows for community incentives to be aligned, resulting in a community driven project
